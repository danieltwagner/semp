**Delete a device**
----
Deletes a device that is registered on the gateway by ID.

* **URL**
    
    /api/devices/:id

* **Method:**
  
  `DELETE`
  
*  **URL Params**

   **Required:**
 
   `id=[string]`

* **Data Params**

* **Success Response:**
  
  * **Code:** 200
 
* **Error Response:**

  * **Code:** 404 NOT FOUND
